【游戏名称】蓝字过关

【游戏介绍】
这是一款H5原生开发的益智类小游戏。游戏中，你需要控制一个小人，通过控制棍子的长短和方向，帮助小人安全地从一个平台跳到另一个平台。游戏操作简单，但需要一定的技巧和判断力。

【玩法说明】
操作：
- 点击屏幕伸出棍子
- 通过点击时长控制棍子的长短
- 棍子会自动从水平方向开始旋转
- 点击屏幕停止旋转并放置棍子
- 小人会沿着棍子走到另一端

道具：
- 小人：玩家控制的角色
- 棍子：连接两个平台的工具
- 平台：游戏中的落脚点

规则：
- 棍子太短无法到达下一平台，小人掉落
- 棍子太长会超出平台边缘，小人掉落
- 需要根据平台距离调整棍子长度
- 成功通过一个平台后会获得分数
- 连续成功会获得连击奖励

目标：
- 帮助小人安全通过尽可能多的平台
- 获得最高分数
- 挑战自己的记录

Tips：
- 观察两个平台之间的距离，估算所需棍子长度
- 棍子会缓慢旋转，适时点击停止
- 练习预估距离的能力
- 连续成功后难度会逐渐增加
- 尽量避免棍子过长或过短

技术介绍：
- 使用HTML5 Canvas和JavaScript开发
- 实现了流畅的触摸控制和旋转动画
- 使用物理引擎模拟角色移动
- 支持移动端优化和响应式设计
- 包含简单的得分系统和连击机制
